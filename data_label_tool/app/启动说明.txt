打开指令：
python app.py xxx.txt
python app.py xxx.json
python app.py xxxxx
三种方式都行。
使用txt时，会检查data目录下是否有对应的json，如果没有会在data目录下创建
使用json时，会直接在那个json文件上浏览、修改、保存
使用编号时，会检查data下是否有对应的json，没有的话打开是空白页

##标注操作说明：
1.会实时保存，无需手动保存。所有的保存修改会在data目录下的json中。
2. 建议先填level2。如果填写level2时level1为空，会自动查找最近相同的level2的level1。比如之前填了level1：下游产业情况，level2：轮胎行业周期性。那么下一次填入level2：轮胎行业周期性 的时候，level1会自动填入“下游产业情况”，就不用多点一次了。
3. 建议把第一句的“二级分类”填入“无关”，这样后面碰到无关的句子可以直接从下拉菜单点击“无关”。
4. "状态“下拉菜单会自动匹配以前填过的同一个二级实体的所有状态，这样输过一次就不用输了。
5. 不可以同时开两个程序，一次只能开一个。标完一个关掉重新开。
